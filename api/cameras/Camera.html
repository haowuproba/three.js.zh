镜头（Camera）

镜头的抽象基类，当创建新的镜头对象时需要继承于此类。

构造函数（Constructor）

[name]()

此构造函数为下面所列的属性设置初始类型，分别为：matrixWorldInverse, projectionMatrix 和 projectionMatrixInverse.

属性（Properties）

.[page:Matrix4 matrixWorldInverse]

This is the inverse of matrixWorld. MatrixWorld contains the Matrix which has the world transform of the Camera.

.[page:Matrix4 projectionMatrix]

This is the matrix which contains the projection.

.[page:Matrix4 projectionMatrixInverse]

This is the inverse of projectionMatrix.

方法（Methods）

.lookAt( [page:Vector3 vector] )

vector — 指向镜头所看位置

设置镜头所看方向为参数向量所指向的位置。当前camera的父节点要为scene或当前camera的空间位置为(0,0,0)。
