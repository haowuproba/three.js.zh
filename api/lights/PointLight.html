点光源（PointLight）

材质为[page:MeshLambertMaterial]或[page:MeshPhongMaterial]所创建的对象将受此光的影响。

Example

var light = new THREE.PointLight( 0xff0000, 1, 100 ); light.position.set( 50, 50, 50 ); scene.add( light );

Constructor

[name]( [page:Float color], [page:Float intensity], [page:Float distance] )

[page:Float hex] — 表示颜色的十六进制数值（RGB）。
[page:Float intensity] — 光照强度。
[page:Float distance] — 光线投射距离。

Properties

.[page:Vector3 position]

光源位置。
默认值 — *new THREE.Vector3()*.

.[page:Float intensity]

光照强度。
默认值 - *1.0*.

.[page:Float distance]

设置为非0值时，光源的强度将从*光源位置*到*distance*线型减弱为0。
默认为 — *0.0*.
