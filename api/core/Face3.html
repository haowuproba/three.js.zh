三角面（Face3）

创建三角面。

Example

var normal = new THREE.Vector3( 0, 1, 0 ); var color = new THREE.Color( 0xffaa00 ); var face = new THREE.Face3( 0, 1, 2, normal, color, 0 );

Constructor

[name]( [page:Integer a], [page:Integer b], [page:Integer c], [page:Vector3 normal], [page:Color color], [page:Integer materialIndex] )

a — A 顶点索引。
b — B 顶点索引。
c — C 顶点索引。
normal — 面法向量或顶点法向量数组。
color — 面颜色或顶点颜色数组。
materialIndex — 材质索引。

Properties

.[page:Integer a]

A 顶点索引。

.[page:Integer b]

B 顶点索引。

.[page:Integer c]

C 顶点索引。

.[page:Vector3 normal]

面法向量。

.[page:Color color]

面颜色。

.[page:Array vertexNormals]

三个顶点的法向量数组。

.[page:Array vertexColors]

三个顶点的颜色数组。

.[page:Array vertexTangets]

三顶点的正切值数组。

.[page:Integer materialIndex]

材质索引（指向 [page:Geometry Geometry.materials]）。

.[page:Vector3 centroid]

面的质心。

Methods

.clone()

创建一份Face3对象的拷贝。
