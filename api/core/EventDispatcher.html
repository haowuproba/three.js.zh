事件派发器（EventDispatcher）

为对象绑定JavaScript特定事件。

代码示例

var Car = function () { EventDispatcher.call( this ); this.start = function () { this.dispatchEvent( { type: 'start', message: 'vroom vroom!' } ); }; }; var car = new Car(); car.addEventListener( 'start', function ( event ) { alert( event.message ); } ); car.start();

构造函数

[name]()

创建eventDispatcher对象，需要调用.call方法将相应的功能函数添加到对象中。

方法

.addEventListener( [page:String type], [page:Function listener] )

type - 添加监听事件的类型。
listener - 事件触发函数。
为相应的事件类型添加监听。

.removeEventListener( [page:String type], [page:Function listener] )

type - 删除监听事件的类型。
listener - 删除的事件函数。
为相应的事件类型删除监听。

.dispatchEvent( [page:String type] )

type - 事件类型。
触发相应事件。
